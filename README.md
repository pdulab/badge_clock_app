# Clock
This a clock app for the [2025 Hackaday Communicator Badge](https://github.com/Hack-a-Day/2025-Communicator_Badge). For now, the goal is to make it look nice, but other features might be added later on (timers, alarms, ...). This is still a work in progress.




## Features
For now the features are :
- Displays the time (it's a clock !)
- Set the time and date manually

Planned features are :
- Show the date (it's not done now, for some reason)
- Synch the time via NTP
- Dark/Light mode (only light mode currently)
- Background images (Tested once, but performance was poor. Might need more troubleshooting.)



## Notes
It is recommended to use a monospaced font. This is not the case of the default font (Montserrat). You can add your own font via this [LVGL tool](https://lvgl.io/tools/fontconverter) (you can use a bpp of 4 and choose the binary file format).
